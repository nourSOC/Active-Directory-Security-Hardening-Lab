# Active Directory Security Hardening Lab

This project demonstrates the implementation of security hardening policies in a Windows Server Active Directory environment.

## Lab Overview
The lab includes deploying a Windows Server domain controller, joining Windows clients to the domain, and applying security policies using Group Policy Objects (GPO).

## Key Implementations
- Active Directory Domain Services deployment
- Domain controller configuration
- Group Policy security hardening
- User privilege restriction
- Endpoint security configuration

## Technologies
- Windows Server
- Active Directory
- Group Policy
- Security Hardening
